Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Lorem ipsum dolor sit amet consectetur adipisicing.

Lorem ipsum dolor sit amet, consectetur adipisicing.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

A tag <u> tem o mesmo efeito visual que a tag <ins>, porém é considerada não semântica e não deve ser utilizada.

Texto sobrescrito

Para inserir coisas do tipo x20+3, use a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O, use a tag <sub>.
